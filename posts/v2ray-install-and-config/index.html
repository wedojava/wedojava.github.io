V2ray install and config

2019.07.15

Reference: https://toutyrater.github.io/prep/install.html

# 1. Install

Install or update by script:

wget https://install.direct/go.sh
sudo bash go.sh

Return:

PORT:111
UUID:9d1712fd-f641-4be9-b2ca-99c9dc000aeb

# 2. Config

2.1. WebSocket+TLS+Web

Stable but not fast!

Server /etc/v2ray/config.json :

{
  "inbounds": [{
    "port": 111,
    "listen": "127.0.0.1",
    "protocol": "vmess",
    "settings": {
      "clients": [
        {
          "id": "9d1712fd-f641-4be9-b2ca-99c9dc000aeb",
          "level": 1,
          "alterId": 64
        }
      ]
    },
    "streamSettings": {
        "network": "ws",
        "wsSettings": {
            "path": "/wahaha"
        }
    }
  }],
  "outbounds": [{
    "protocol": "freedom",
    "settings": {}
  }]
}
