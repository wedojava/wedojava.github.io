Method 1.1

git reset --hard HEAD^        回退到上个版本
git reset --hard commit_id    退到/进到 指定commit_id
git push origin HEAD --force  把回退的某个版本提交的远程

Method 1.2

# 强制推送到远程分支, 会抹去远程库的提交信息, 不要这么干
